# 🧩 Legado Azul: Inteligência de Dados e o Futuro do Cuidado

Este repositório contém o **MVP (Minimum Viable Product)** desenvolvido para a pós-graduação em **Ciência de Dados e Analytics pela PUC-Rio**.

## 🚀 Sobre mim
Especialista em **Finanças & Tesouraria** e Pós-graduanda em **Ciência de Dados e Analytics** pela PUC-Rio. Minha trajetória de 4 anos no setor financeiro agora se une ao poder da análise de dados para gerar insights estratégicos e automação de processos.

### 🛠️ Foco de aprendizado:
- **Data Science:** Desenvolvimento de modelos preditivos e análise exploratória (Python, Pandas, Scikit-Learn).
- **Fintech & Tesouraria:** Interseção entre gestão de caixa, compliance e tecnologia.
- **Business Intelligence:** Transformação de dados brutos (SAP, OMIE, Milenium) em dashboards decisivos.

### 📚 Principais Competências:
- **Linguagens:** Python (com foco em Data Science).
- **Bibliotecas:** Pandas, Matplotlib, Seaborn, Scikit-Learn.
- **Finanças:** Fluxo de Caixa, Auditoria, Gestão de Garantias e Riscos.
- **Sistemas:** SAP, OMIE, Milenium e Excel Avançado.

### 🥋 Além do código:
Fora do mundo dos dados e das planilhas, busco disciplina e foco no **Jiu-Jitsu**, trazendo a mentalidade de resiliência do tatame para os desafios do ambiente corporativo.
